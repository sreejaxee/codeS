ROCK PAPER SCISSOR

Experience the excitement of Rock Paper Scissors brought to life. CURRENCY CONVERTER

I developed a responsive currency converter using HTML, CSS, and JavaScript, drawing inspiration from existing projects while implementing my unique design and functionality. Featuring a clean interface with intuitive controls, users can input an amount, select currencies from flag-icon dropdowns, and instantly calculate conversions using real-time exchange rates fetched from an API. The converter's design ensures accessibility across devices, leveraging dynamic flag icons and interactive elements for a seamless user experience. Explore this project in my portfolio to see how I merged creativity with practical frontend development skills to deliver a user-friendly financial tool.

TIC TAC TOE

I developed a dynamic Tic Tac Toe game entirely using HTML, CSS, and JavaScript. Players can choose their symbol (X or O) and play against the computer. The game includes features such as disabling cells after a winner is declared, preventing the computer from making moves post-victory, and restarting the game seamlessly from a popup interface. This project showcases my proficiency in front-end development, interactive web design, and JavaScript programming.
